ParrotCrawl

Welcome to the ParrotCrawl the only web crawler inspired by the Cult of the Parrot.

To use ParrotCrawl enter a URL you would like to start from and the crawler will search outward from that webpage navigating through each link. If you would like the ParrotCrawl to search for a string on the pages it visits enter it in the optional field. You will also need to choose a value of n which will determine how many links from the original URL that the search will continue for. You must also choose a style of search, either depth first or breadth first. The depth first search will choose links randomly and continue until n is reached. The breadth first search will navigate through a limited number of links on each webpage and then navigate through the links on each of those pages until the depth of n is reached. If you choose to use breadth first please note that for each value of n a different number of links per webpage searched will be returned. For n = 1 the max is 1000 links returned from the URL you have given. For n = 2 the max is 30 links returned per webpage navigated to. For n = 3 the max is 10 links returned per webpage navigated to. For n = 4 the max is 5 links returned per webpage navigated to. To start click the 'get started' button!
